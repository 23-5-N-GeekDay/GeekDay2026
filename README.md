# 关于北归 / About 23.5°N GeekDay

**“北回归线极客节”（23.5°N GeekDay）** 是粤东首个大型极客盛会。“北回归线” 取名自潮汕地区所处的北纬 23.5 度，坐落在汕头的两座北回归线标志塔是本土青年成长记忆中的标志性建筑。活动简称“北归”，也蕴含了主创团队**在京学成、自北归南、反哺家乡**的初衷。

“北归极客节”自2024年开始筹备，首届以一场 Hackathon 黑客马拉松、一场 Nerd Bar 圆桌酒会、一场 Maker Show 展览组成，计划长期持续运营。北回归线是太阳在地球上直射的北界，也是创意与热情的沸点。我们期待同好们用实践和创造开拓认知边界，以灵感的迸发、智慧的涌现建设潮汕，启迪湾区，直至触达世界。我们也期待共同的极客精神超越方言、地域，持续促成极客联结与思想交融，在现代化、智能化的阈限中建构创造价值认同与文化互认体系。
